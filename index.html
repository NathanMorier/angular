General notes on Angular:

  • While it's generally better to put JS at the end of the body, you want Angular to be placed in the head.

Some notes on MCV:

  • MODEL is the data, which can be a JSON file, database data, etc.
  • VIEW is used to display content & data in browser
    • Use Expressions to insert data into views: <html tag> {{ data }} </html tag>
  • CONTROLLER is like the bridge between the MODEL and the VIEW, controlling the interaction, like a button click for example.
  • You may hear of MCV refered to MCW, the 'W' meaning 'Whatever'. The underlying principle exactly much the same, just a separation of concerns.
